子弹笔记学习概要三

内容说明：

该笔记整理于杉野干人-《子弹笔记术》。图片，案例均来自该著作，个人整理仅供分享交流

不写正确的废话

  1. 加入市场部的小A要在早会作一次简短的表明信念的演讲，需要准备子弹笔记式的PPT.

  2. 听众： 市场部的前辈，他们带着考察新人能力的心态来参加早会，想看看今年的新人里有没有比较能干的。

  3. 小A写的子弹笔记PPT(不合格)：

  4. 结果：

    • 市场部的前辈们是一群精通市场营销并且经验丰富的实干精英。所以，他们很可能会觉得“这个演讲者不中用”。

    • 这份子弹笔记里没有新信息。小A写的这6条保证，乍一看有模有样的，但只是形式上有模有样而已。

    • 他写的都是理所应当的事情，是所有在市场部工作的员工都在追求的目标.

    • 小A写的子弹笔记无法打动前辈，因为没有需要他们重新进行信息处理的内容。

一开始就要表明立场

  1. 对信息接收者来说，子弹笔记的内容要有信息处理价值，并且能打动人心.

  2. 为了打动对方、使其按我们的想法行动而斟酌词句，我们把这个过程称为“信息化”.

  3. 实现信息化的必要条件是什么呢？

    • 表明立场。表明立场就是把自己的态度明确说出来——是赞成还是反对，是采用A方案还是B方案，是想要做这件事还是不想做。

    • 在表明自己的立场时，也有几个技术性的诀窍:排除隐藏性重复内容、使用否定、使用数字这三个诀窍等。

诀窍一：排除“隐藏性重复内容”

  1. 重复内容：就是像“给脸洗一下脸”、“头痛很痛” 这种含义重复的表达方式。

  2. 隐藏性重复内容：是指文章里没有重复的语句，但是表达的某些内容是重复的，没有特地写出来的必要，也没有太大的意义。

  3. 隐藏性重复内容具体是指什么样的内容呢？

  4. 隐藏性重复内容只是把理所应当的事情说得冠冕堂皇。对信息的接受方没有任何意义。

看似有用的无用词

  1. 排除隐藏性重复内容，对企划案实现信息化十分有效，因为:

    • 企划案是由信息表达者主导的表达方式，其节奏也由信息表达者决定。

    • 每一个句子是否干净利落是成败的关键。

  2. 企划案里经常出现的隐藏性重复内容都有哪些呢？

  • 其中很多是邮件和备忘录等其他表达方式中也经常会出现的内容，但在企划案里出现得更为频繁。

  • 在充斥着唯心论的企业方针说明和经营战略发布中，我们经常可以看见各种隐藏性重复内容。

    • 如果是长期目标，考虑到时间跨度很长，是有必要重申这种理所当然的内容的。

    • 中短期经营方针来说，战略性很重要。

    • 在用语言来表现战略性的时候，企业需要清晰地表明自己的立场。

    • 如果一个企业的经营方针里全是含有隐藏性重复内容的句子，就说明这个企业存在问题：

      • 要么是这个企业的经营方针里根本就没有战略，

      • 要么是企业有战略但是因为内容不具备表现力，所以企业的各部门不会积极行动。

      • 也可能是这个企业将制定经营战略这件事，交给了有官僚主义想法的员工。这些员工因为害怕清晰地表明立场，转而选择保守的表达方式。

柔软的否定：比起A来B更好

  1. 用相对否定，即强调“通过比较，选择更好的那一方”这种表达方式，我们可以通过比较来体现出否定的语气从而表明立场。

  2. 比如，“比起A来B更好”这句话，用B比A更好来表明立场，但未必否定了A。

  1. 用“从A变成B”的句式来代替简单的“变成B”的句式，通过加入原本的情况来形成比较，体现出否定语气。

  2. “从A变成B”这种通过比较来体现否定，从而使子弹笔记实现信息化的案例:

  3. 不想使用绝对否定的人，可以通过比较句式来表达否定。这种方式既缓和了语气，也能够清晰地表明我们的观点。

诀窍三：把形容词和副词换成数字

  1. 在叙述“大小”等表示程度的内容时，使用数字后，内容会表达得更清晰。

  2. 使用形容词和副词来表达程度，但是如果只用形容词和副词，这个“程度”就会因对方的解读不同而产生不同的答案。

  3. 将形容词和副词这种模糊的表达，换成明确的数字，对方就很容易理解你想表达什么了。

  5. 点评：

    • 在第一句话里，“许多”这个形容词可以换成数字，比如，换成“大约1 000位”，对方可能就会觉得“这个人的志愿活动坚持
      了这么长时间。

    • 还有一个小技巧，不使用“大约”这种模糊的词语，而是用具体数字来表达，比如“1 047位”。因为像“大约1 000位”这种
      表达，很可能会出现在别的学生的自我介绍中。

    • 在第二句话中，“许多”和“巨大的”这两个词也可以换成数字:

      • 比如，将“许多”换成“16家”。
      • 将“巨大的”换成“260万日元”。

用子弹笔记武装你的简历

  1. 海外简历中的子弹笔记

  1. 尤其是“掌控的资金数额”、“担任管理职务，所负责团队的人数”、“营业额和盈利等金额”这三项数据，可以被视为海外简历中的三大数字。

  2. 子弹笔记一般是按照时间顺序从近到远来书写。

  3. 通过使用数字来使自己想要表达的内容变得清晰明确的，不只是简历，学生发给企业的应聘申请表也是如此。

  4. 很多学生对自己的闪光点并不清楚。这时不妨通过数字进行思考。

    • 如何将自己的经历和经验数字化呢？这些数字适合运用于哪些指标呢？

  5. 将想要表达的内容用数字来进行思考，可以帮助我们重新审祝自己拥有的闪光点。(个人觉得，数字化依赖于个人平时的记录
    和统计，而不是编造数字)

不提“状态”和“期限”的目标都是假目标

  1. 对于目标来说，数字是不可或缺的。

  2. 为什么说:大多数子弹笔记都不存在能够被称为目标的功能要素，它们仅仅是文字的罗列?

    • 因为这些子弹笔记里根本没有使用数字。

  3. 为了让子弹笔记清晰地描绘出企业未来的样子，就必须明确这是“何时”的目标，需要做到“怎样”的程度。

    • 思考：在制定自己的工作计划，工作目标时是否达到了这种要求。

  4. 目标必须要有两个数字，一个表示状态，另一个表示期限。如果难以明确地用数字表达，用表示程度的词语，或者能够判断是否已经
    实现的表达方式也可以。

    • 截至2030年，在全世界30个国家开展商业活动

    • 在5年后成为全球范围内的业界第一

  1. 个人的目标：

    • 比起“总有一天要成为写小说的作家”，“5年后，要拿到文学杂志的新人奖”这个目标更加具体。

    • 那些说着“想要学好英语”，“有时间的话就学习”的人，到最后其实什么没也做。

    • 明确表示“3年后，托业考试要拿到950分”的人，肯定在为自己的梦想努力着。

真诚的子弹笔记最动人

  1. 冠冕堂皇的子弹笔记。这种子弹笔记所写的内容看起来那么理所当然，任谁也无法反驳。

    • 比如，在听到“尽自己最大努力，高效率地完成工作”这句话时，谁也不会想要反驳。

    • 冠冕堂皇的子弹笔记，就成了那些想要 应付 一切的人的“武器”。

  2. 如果只是表达理所应当的事情，那么从一开始就没有表达的必要。
